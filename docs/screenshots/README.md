# Screenshots

Add app screenshots here for the main README. Suggested filenames:

- **`loom-main-view.png`** — Full app: sidebar (file list), chart view with a visualization, right panel (Stats/Chart/Export tabs), and preview footer.
- **`loom-data-sources.png`** — Data & sources panel: “Choose folder” and “Recent CSV datasets” from Data.gov with Save to folder.

To capture:

1. Run `make spin` and open a folder (or use demo data in the browser with `make thread`).
2. In Chart view, select a file and pick a chart suggestion.
3. Take a full-window or region screenshot and save as `loom-main-view.png`.
4. Open **Data & sources**, wait for Data.gov list (Tauri only), then capture as `loom-data-sources.png`.

Use PNG for clarity. The main README references these paths.
